# 道具UI设计提示词

## 一、整体风格定位

### 推荐风格：卡通扁平像素风

**风格特点**：
- 简洁明快的线条
- 鲜艳饱和的色彩
- 圆润可爱的造型
- 适合休闲游戏

### 像素规格

| 道具类型 | 推荐尺寸 | 格式 | 说明 |
|---------|---------|------|------|
| 道具图标 | 64x64 px | PNG透明 | 游戏内显示 |
| 道具图标@2x | 128x128 px | PNG透明 | 高清显示 |
| 道具图标@3x | 192x192 px | PNG透明 | 超高清显示 |

## 四、AI生成工具推荐

### 在线工具

| 工具 | 网址 | 特点 |
|-----|------|------|
| Midjourney | midjourney.com | 高质量，付费 |
| DALL-E 3 | openai.com/dall-e-3 | 简单易用 |
| Leonardo.ai | leonardo.ai | 免费，游戏风格 |
| Bing Image Creator | bing.com/create | 免费，DALL-E驱动 |

### 提示词优化技巧

1. **添加风格词**: flat design, kawaii, cartoon, game icon
2. **指定尺寸**: 64x64 pixels, square format
3. **指定背景**: transparent background, no background
4. **添加质量词**: high quality, clean lines, vector style
5. **避免词**: realistic, photo, 3D, complex details

---

## 五、生成后处理

### 必要处理

1. **去除背景**: 使用 remove.bg 或 Photoshop
2. **调整尺寸**: 确保为 64x64、128x128、192x192
3. **优化颜色**: 确保颜色鲜艳饱和
4. **添加描边**: 可选，增加识别度

### 工具推荐

- **去背景**: remove.bg, Photoshop, GIMP
- **调整尺寸**: Photoshop, GIMP, Photopea
- **批量处理**: TinyPNG, ImageOptim

## 六、完整示例

### 护盾道具完整提示词

```
Prompt:
A cute cartoon shield icon for mobile game, 64x64 pixels, flat design style, 
rounded corners, blue gradient (#2196F3 to #1976D2), white highlight on top, 
subtle drop shadow, game UI element, transparent background, simple and clean, 
kawaii style, suitable for casual game, icon design, vector-like quality,
high quality, clean lines, no background

Negative Prompt:
realistic, photo, 3D, complex details, shadow, gradient background, 
text, watermark, low quality, blurry
```

### 批量生成建议

使用 Leonardo.ai 或 Midjourney 的批量功能：
1. 一次性生成所有道具图标
2. 保持风格一致性
3. 统一调整尺寸和格式

## 七、验收标准

| 检查项 | 标准 |
|-------|------|
| 尺寸 | 64x64 px (±2px) |
| 格式 | PNG-24 透明 |
| 文件大小 | < 10KB |
| 风格 | 卡通扁平，风格统一 |
| 颜色 | 鲜艳饱和，对比度高 |
| 边缘 | 清晰锐利，无锯齿 |
| 背景 | 完全透明 |
